A modern all-in-one financial calculator web application built using HTML, TailwindCSS, and JavaScript.
It provides multiple financial and business calculation tools in a single interactive dashboard.

The interface uses a navy blue theme with glowing yellow UI elements and smooth animations to deliver a clean and modern user experience.

This project is implemented as a single-page application (SPA) where calculators load dynamically based on tab selection.
